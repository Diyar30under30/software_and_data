# 🎓 School Management System  
A desktop application built with **Python (tkinter)** and **SQLite** for managing school-related data such as teachers, courses, and student registrations.  
This project was developed as part of the **Z02DT – Software and Databases** module at Coventry University Kazakhstan.

---

## 📌 Features

### ✔ Core Functionality
- **Teacher Management**
  - Add new teachers  
  - Update existing records  
  - Delete teachers  
  - View full teacher list  

- **Course Management**
  - Create courses  
  - Edit course details  
  - Remove courses  
  - Display course catalog  

- **Registration Management**
  - Register students for courses  
  - View registrations  
  - Delete or modify registration details  

- **Database**
  - SQLite-based local storage  
  - Proper table structure with foreign keys  
  - SQL queries for CRUD and reporting
